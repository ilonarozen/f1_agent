# F1 Agent 🏎️

Python script that fetches real-time F1 driver data using the OpenF1 API.

## What it does
- Calls the OpenF1 API to retrieve current F1 drivers
- Displays driver name, team, and nationality
- Handles connection errors and HTTP errors gracefully

## How to run
pip install requests
python f1_results.py

## Tech used
- Python 3
- Requests library
- OpenF1 API
